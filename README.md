Assets created in Blender for Games and Archviz.  
Note: This repo uses textures from https://github.com/stlavleen/Textures. 
If there are issues with textures (objects of purple color in viewport shading mode) in Blender, do the following:
1. Clone Textures repo.
2. In Blender: Go to File -> External Data -> Find missing files. Then select Textures folder of corresponding project in Textures repo. Alternatively, you can manually set correct textures without auto-search.
